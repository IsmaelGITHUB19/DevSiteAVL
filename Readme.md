Projeto avaliativo Beck end 1 em PHP
Requisitos - Avaliação Final - Sistema.

O sistema será utilizado como a avaliação final do semestre. Portanto, destaco alguns requisitos que o sistema deverá atender:
5 CRUDs;
Relacionamento pelo menos três tabelas.
Deverá conter autenticação, login e senha - com cadastro de usuários;
Simulação de uma venda, que deverá conter informações do produto, como por exemplo: código, descrição e valor,  dados do cliente, e valor da venda contendo a  condição de pagamento. Não precisa controlar o estoque.
elo menos 5 relatórios.
